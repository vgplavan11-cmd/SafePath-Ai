**SafePath AI – Smart Women Safety & Navigation Assistant**

SafePath AI is a modern web application designed to enhance personal safety using intelligent, real-time features. The application combines emergency response tools with location-based safety analysis to help users make safer decisions during travel and daily activities.

The system includes a Fake Call feature that simulates incoming calls to help users escape uncomfortable or unsafe situations. It also provides a dynamic safety map that visualizes safe and risky areas based on environmental factors such as lighting, crowd density, and time conditions.

A key highlight of the application is the Safest Route Finder, which suggests routes based not only on distance but also on safety levels, offering users a more secure navigation experience. The app also integrates voice recognition, allowing users to trigger emergency actions using keywords like “help” or “danger.”

Additionally, the Virtual Guardian Mode actively monitors users during their journey. If the user becomes unresponsive after a set time, the system automatically triggers safety measures such as live location tracking and emergency alerts.

Built with a modern lavender-themed UI, SafePath AI focuses on both usability and aesthetics, delivering a clean, intuitive, and mobile-friendly experience.

**Key Features:**

* Fake Call Simulation for emergency situations
* Safe/Risky Area Visualization with map integration
* Safest Route Recommendation system
* Voice-activated emergency detection
* Real-time GPS live location tracking
* Virtual Guardian Mode for active safety monitoring

SafePath AI demonstrates how technology and intelligent design can be used to create practical, user-focused safety solutions for real-world challenges.
